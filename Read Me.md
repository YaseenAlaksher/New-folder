# Yaseen Portfolio Website

A modern and responsive personal portfolio website built to showcase my skills, projects, and journey as a Front-End Developer.

## 🚀 Live Demo

[View Portfolio](https://yaseenalaksher.github.io/My-Portfolio-main/)

---

## 📌 About The Project

This portfolio was designed and developed to present:

* My Front-End development skills
* Featured projects
* Current learning path
* Contact information
* Professional online presence

The website focuses on clean UI, responsive design, modern layouts, and smooth user experience.

---

## ✨ Features

* Fully Responsive Design
* Modern Dark Theme UI
* Smooth Scrolling Navigation
* Animated Hover Effects
* Intersection Observer Reveal Animations
* Project Showcase Section
* Skills & Technology Stack Section
* Contact Information Section
* Mobile-Friendly Layout
